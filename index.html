6 Degrees of Seperation Application

This project is an application utilizing Python, C++, and the Instagram API to test the theory that everyone is connected by no more than 6 degrees of seperation. We are utilizing a web scraper and the Instagram API to get a user, a target account and then go through their followers to determine if that user and the target are connected. We acheive this by utilizing a dual Breadth First Search algorithm to count the number of connections and and connect the two accounts. We are currently in the process of developing this technology.
